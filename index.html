What makes ROOMR different

Built to vanish
when the work is done.

⏳
Time-bounded rooms

Set a duration from 30 minutes to 7 days. When the clock hits zero, the room and everything in it disappears automatically.

💬
Live chat

Real-time messaging synced across all room members via BroadcastChannel. Chat, share, collaborate — all in context.

📁
Organized file sharing

Create folders, upload files, preview images and documents inline. Drag-and-drop support with folder hierarchy.

🔐
Admin controls

Room creators control everything — who can upload, who can delete, how long the room lives, and when to kill it.

🔗
One-click invites

Share a 6-character room code or a direct link. Anyone can join in seconds — no setup required for guests.

🌑
Zero residue

No permanent storage. No logs. No data harvesting. Your session disappears completely — by design.
